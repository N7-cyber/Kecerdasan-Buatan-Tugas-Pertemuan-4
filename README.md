# Kecerdasan-Buatan

## Tugas 2: Pencarian Jurnal Semantic Network

Berikut adalah detail referensi jurnal yang menggunakan metode jaringan semantik (*Semantic Network*):

* **Judul Jurnal:** Implementasi Jaringan Semantik dengan Hypermedia
* **Penulis:** M. Candiasa
* **Tahun:** 2005
* **Sumber/Penerbit:** Jurnal Pendidikan Teknologi dan Kejuruan (Vol. 2, No. 1)

---
*Catatan Tambahan tentang Jurnal:*
Jurnal ini berfokus pada implementasi *semantic network* dalam perancangan arsitektur hypermedia untuk menyusun struktur informasi agar dapat merepresentasikan pengetahuan secara lebih terstruktur dan efisien.
